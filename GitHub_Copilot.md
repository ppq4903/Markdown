# 1、

大约60%的恶意软件都能按预期运行。

最容易创建的恶意软件是**间谍软件 Spyware**。它可以偷偷激活用户的网络摄像头和麦克风，将录音保存到用户公共文档中的伪装文件中，并在一分钟内执行。例如，视频文件保存为1670194934.8706307，其中包含用户的记录。

**DoS程序**是第二大成功的恶意软件。生成的脚本会阻止人们在笔记本电脑上做任何事情，要么每毫秒将鼠标移动到屏幕左上角，要么创建一个无法最小化或退出的屏幕，因为它不断被调用到屏幕的前面。

**Ransomware**对文件进行了加密，**wiperware**覆盖并删除了它们。生成的特洛伊木马执行了特洛伊木马的基本功能，因为它们成功打开了文档并在其中注入了代码，但当打开被篡改的文件时，代码并没有按照promts中定义的那样执行。蠕虫是生成的最不成功的恶意软件类别，这可能是因为它们通常是网络爬虫，比本研究中创建的其他恶意软件复杂得多。

**在VirusTotal上，平均只有7/72的供应商将这些脚本标记为恶意**

- 其创建的恶意软件缺乏活跃的入站和出站网络连接
- 生成的恶意软件脚本相对简单，没有使用现实世界恶意软件中常见的复杂混淆或规避技术
- 生成的代码可能看起来是良性的，因为它来自公共代码库。
